## Requirements

To run the project you need these build tools:

* node 20+
* hugo 39+

## Usage

Clone the repo.

```bash
git clone git@github.com:Mint-System/Website.git
```

Optionally install the vercel cli and login.

```bash
task install
```
### Build

Build the website.

```bash
task build
# or
task vercel-build
```

Serve from public directory.

```bash
task serve
```

### Develop

Install hugo and run local server.

```bash
task dev
# or
task vercel-dev
```
